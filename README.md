Gears
-----
>Gears feature tests using uJson and Scala 3, targeting the JVM ( not Scala Native ).

Note
----
>If using Scala Native, **sbt run** fails to link **AsyncApp** on **java.net.URL**.

Build
-----
1. sbt clean compile

Run
---
1. sbt run
